What to install offline

If when calling /ingest/ocr you see errors like “OCR: Pietesseract and Pilov are required”, “PDF requires PDF2ymage/poppler”, install the dependency locally.

Windows

choco install tesseract
# Popolnite PATH tesseract'a i poppler'a (bin)
pip install pytesseract pillow pdf2image

Skachayte poppler for Windows and add it bin v PATH.

Linux (Debian/Ubuntu)

sudo apt-get update
sudo apt-get install -y tesseract-ocr poppler-utils
pip install pytesseract pillow pdf2image

macOS

brew install tesseract poppler
pip install pytesseract pillow pdf2image

Proverka

python -c "import pytesseract; from PIL import Image; import pdf2image; print('OK')"

Why is this important

Without these packages, OCD for PDF/images will not work (modules modules/ingest/ocr_engine.py kidayut RuntimeError with explanation). This operatsionnaya dependency and it is installed on the host where Esther is running.
