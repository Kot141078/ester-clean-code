Chto ustanovit offlayn

Esli pri vyzove /ingest/ocr vy vidite oshibki vida «OCR: trebuetsya pytesseract i Pillow», «dlya PDF nuzhen pdf2image/poppler», ustanovite zavisimosti lokalno.

Windows

choco install tesseract
# Popolnite PATH tesseract'a i poppler'a (bin)
pip install pytesseract pillow pdf2image

Skachayte poppler dlya Windows i dobavte ego bin v PATH.

Linux (Debian/Ubuntu)

sudo apt-get update
sudo apt-get install -y tesseract-ocr poppler-utils
pip install pytesseract pillow pdf2image

macOS

brew install tesseract poppler
pip install pytesseract pillow pdf2image

Proverka

python -c "import pytesseract; from PIL import Image; import pdf2image; print('OK')"

Pochemu eto vazhno

Bez etikh paketov OCR dlya PDF/izobrazheniy ne zarabotaet (moduli modules/ingest/ocr_engine.py kidayut RuntimeError s poyasneniem). Eto operatsionnaya zavisimost i ona stavitsya na khoste, gde krutitsya Ester.
