# Profiles

Practical profiles for local-first `c` runtime and operational lower bounds.

## Documents

- `MINIMUM_C_NODE_PROFILE_v0.1.md`
- `pdf/Minimum_C-Node_Profile_v0.1_academic.pdf`

This profile is not a commercial product spec, legal regime, rights charter, or final implementation standard.

It maps ontology into runtime surfaces: local raw custody, memory, permissions, witness, budgets, oracle routing, backup/restore, stop/freeze, and human-readable control.
