# Sustainability Kit - ready-made materials for green agents

**What's inside**
- Checklists (marketing, events, SMB audit).
- Bazovye koeffitsienty/faktory CO₂e v `metrics.json`.
- Sovmestimye plany kaskada v `examples/plans/*sustainability*`.

**Mosty**
- Yavnyy: (Dokumentatsiya/Dannye ↔ Mysli/Deystviya) - eksheny `sustainability.kit.*` otdayut eti materialy cherez `/thinking/act`.
- Skrytyy #1: (UX ↔ Kaskad) — plany mozhno gonyat iz suschestvuyuschey paneli kaskada (admin_cascade.js) :contentReference[oaicite:5]{index=5}.
- Skrytyy #2: (UX ↔ RuleHub) - YAML-politiki redaktiruyutsya cherez `/rulehub/config` (kak v admin_rulehub.js) :contentReference[oaicite:6]{index=6}.

**Zemnoy abzats**  
Eto polka s "bystrymi pobedami": dostaete checklist pod zadachu, podtverzhdaete metrikami, formiruete plan - i Ester srazu prevraschaet znaniya v deystviya, ne menyaya API.
