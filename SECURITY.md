# Security Policy

## Reporting a Vulnerability
- Use GitHub private vulnerability reporting for this repository.
- Do not post exploitable details in public issues.
- Include reproduction steps, impact, and affected version or commit.

## Response Targets
- Initial triage: within 5 business days
- Remediation plan: after triage confirms validity
- Disclosure: coordinated after a fix or mitigation is available

## Scope
This policy covers the code and configuration in this repository. Trademark misuse is handled under `TRADEMARK.md`.
